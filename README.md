## ⚡ Getting Started

### First Time Setup

1. **Download** Ember for your operating system
2. **Install** or run the application
3. **Connect** to your Ember server or join with an invite link
4. **Create** your account and save your recovery code
5. **Start chatting** securely!

### Joining a Server

**With Invite Link:**
- Click an `ember://invite/` link
- Ember will open automatically and connect you

**Manual Connection:**
- Open Ember
- Enter your server address
- Log in or create an account

## 🔐 Security & Privacy

### Your Recovery Code

When you first create an account, Ember gives you a **16-digit recovery code**:

```
XXXX-XXXX-XXXX-XXXX
```

**⚠️ IMPORTANT:** Save this code somewhere safe! You'll need it to:
- Restore your account on a new device
- Recover access if you forget your password
- Move between computers

**Where to store it:**
- Password manager (recommended)
- Written down in a secure location
- Encrypted digital file

### Encryption Explained

- **End-to-End Encryption**: Messages are encrypted on your device before sending
- **Zero-Knowledge**: Server administrators cannot read your messages
- **Secure Storage**: Your keys are protected by your operating system
- **No Tracking**: Ember doesn't collect or sell your data

## 💾 Backup & Recovery

### Account Recovery

If you lose access to your device:

1. **Install Ember** on your new device
2. **Click "Recover Account"**
3. **Enter your 16-digit recovery code**
4. **Set a new password**

### Data Backup

Ember stores your data locally. To back up:

**Linux:**
```
Copy ~/.config/ember-client/ to a safe location
```

**Windows:**
```
Copy %APPDATA%\ember-client\ to a safe location
```

## 🔧 Troubleshooting

### Common Issues

**Can't Connect to Server**
- Check your internet connection
- Verify the server address is correct
- Ensure the server is online

**Voice/Video Not Working**
- Grant microphone/camera permissions
- Check your device settings
- Try different audio/video devices

**Forgot Password**
- Use your 16-digit recovery code
- Click "Recover Account" on the login screen

**App Won't Start**
- Restart your computer
- Check for system updates
- Try reinstalling Ember
